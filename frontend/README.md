# TeslaCharge Frontend

## Description
Frontend application for Tesla charging station reservation system.

## Technology Stack
- **HTML5** - Markup
- **CSS3** - Styling (Tailwind CSS)
- **JavaScript (ES6+)** - Client-side logic
- **Font Awesome** - Icons

## File Structure
- `index.html` - Main application page
- `app-simple.js` - Main application logic
- `manifest.json` - PWA manifest

## Features
- ✅ OTP verification interface
- ✅ Vehicle registration form
- ✅ OCR scanner simulation
- ✅ Station dashboard
- ✅ Reservation system
- ✅ Receipt printing
- ✅ Responsive design

## Components
- **Authentication**: Email/phone OTP verification
- **Vehicle Management**: Manual and OCR registration
- **Station Dashboard**: Available charging stations
- **Reservation Form**: Complete booking system
- **Receipt System**: Print functionality

## Usage
1. Open `index.html` in a web browser
2. Connect to the backend server at `http://localhost:3000`
3. Use test email: `test@tesla.com` with OTP code

## Design
- **Tesla-inspired**: Red and black color scheme
- **Glass morphism**: Modern UI effects
- **Responsive**: Works on all devices
- **Accessible**: Semantic HTML and ARIA labels

## API Integration
- Base URL: `http://localhost:3000/api`
- Authentication: JWT tokens
- Error handling: User-friendly messages
- Loading states: Visual feedback
